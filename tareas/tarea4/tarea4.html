Ejercicios 25 y 27: valor esperado de funciones de variables aleatorias independientes.

📐 Ejercicio 25 — Área Esperada del Rectángulo

Un topógrafo quiere delimitar un cuadrado de lado L. Por error mide un rectángulo con:

  • lado norte-sur X
  • lado este-oeste Y

Donde X y Y son independientes y X, Y ~ U[L−A, L+A]

Se pide: el área esperada del rectángulo.

1

Área del rectángulo

El área es el producto de los dos lados:

A = X · Y
Queremos: E(A) = E(XY)
2

Usar independencia

Si X y Y son independientes, la esperanza del producto es el producto de esperanzas:

E(XY) = E(X) · E(Y)
3

Esperanza de una distribución uniforme

Para X ~ U(a, b), la media es:

E(X) = (a + b) / 2

Aquí a = L − A y b = L + A, entonces:

E(X) = ((L − A) + (L + A)) / 2
E(X) = 2L / 2
E(X) = L

Igualmente, E(Y) = L

4

Área esperada

E(XY) = E(X) · E(Y)
E(XY) = L · L
E(XY) = L²
Resultado Ejercicio 25
E(A) = L²
💡
Interpretación: En promedio, el área del rectángulo medido coincide exactamente con el área del cuadrado deseado (L²). El error de medición no sesga el resultado esperado, ya que las desviaciones se cancelan en promedio.

⏱ Ejercicio 27 — Tiempo Esperado de Espera

Annie y Alvie acuerdan encontrarse para desayunar entre 12:00 y 1:00 PM.

  • X = hora de llegada de Annie
  • Y = hora de llegada de Alvie

Las variables son independientes con densidades:

fₓ(x) = 3x²    0 ≤ x ≤ 1
f_Y(y) = 2y    0 ≤ y ≤ 1

Se pide: ¿Cuál es el tiempo esperado que la primera persona en llegar debe esperar?

Sugerencia: h(X,Y) = |X − Y|

1

Variable que queremos calcular

El tiempo de espera es la diferencia absoluta entre las horas de llegada:

E(|X − Y|)
2

Densidad conjunta (por independencia)

f_{X,Y}(x, y) = fₓ(x) · f_Y(y)
f_{X,Y}(x, y) = 3x² · 2y
f_{X,Y}(x, y) = 6x²y
3

Integral de esperanza — dividir en dos regiones

E(|X − Y|) = ∫₀¹ ∫₀¹ |x − y| · 6x²y dx dy

Región 1: x > y

∫₀¹ ∫₀ˣ (x − y) · 6x²y dy dx
Resultado = 1/6

Región 2: y > x

∫₀¹ ∫₀ʸ (y − x) · 6x²y dx dy
Resultado = 1/12
4

Resultado final

E(|X − Y|) = 1/6 + 1/12
E(|X − Y|) = 2/12 + 1/12
E(|X − Y|) = 3/12
E(|X − Y|) = 1/4
Resultado Ejercicio 27
E(|X − Y|) = 1/4 hora = 15 minutos
💡
Interpretación: El tiempo esperado que la primera persona en llegar debe esperar es 15 minutos. El intervalo total es 1 hora, y dado que Annie llega con distribución cúbica sesgada hacia el final y Alvie con distribución triangular, en promedio la espera es un cuarto de hora.

📊 Resumen de Resultados

25
Área esperada del rectángulo
E(A) = L²

El área promedio coincide con la del cuadrado deseado.

27
Tiempo esperado de espera
E(|X−Y|) = ¼ hora

En promedio, la primera persona espera 15 minutos.
