1. Distribución Geométrica

¿Qué modela?

El número de intentos necesarios hasta obtener el primer éxito.

Propiedades
  • Ensayos independientes
  • Probabilidad de éxito constante p
  • Solo dos resultados: éxito o fracaso
  • Proceso sin memoria
Variable aleatoria

X = número de intentos hasta el primer éxito

Función de probabilidad
P(X = k) = (1 − p)^(k−1) · p
donde k = 1, 2, 3, ...
Ejemplo simple

Lanzar una moneda con probabilidad de cara p = 0.5 hasta obtener la primera cara.

2. Distribución Exponencial

¿Qué modela?

El tiempo entre eventos en un proceso continuo.

Propiedades
  • Eventos independientes
  • Tasa constante λ
  • Distribución continua
  • Propiedad sin memoria
Variable aleatoria

X = tiempo hasta el siguiente evento

Función de densidad
f(x) = λ · e^(−λx)    con x ≥ 0
Ejemplo simple

Tiempo entre llegadas de clientes con tasa λ = 0.5.

f(x) = 0.5 · e^(−0.5x)
Distribución Exponencial (λ = 0.5)

3. Distribución de Poisson

¿Qué modela?

El número de eventos que ocurren en un intervalo fijo de tiempo o espacio.

Propiedades
  • Eventos independientes
  • Tasa promedio constante λ
  • Eventos raros pero posibles en cualquier momento
Variable aleatoria

X = número de eventos en un intervalo

Función de probabilidad
P(X = k) = (e^(−λ) · λ^k) / k!
Ejemplo simple

Número de clientes que llegan por hora. Promedio: λ = 3

4. Distribución Binomial

¿Qué modela?

El número de éxitos en un número fijo de ensayos independientes.

Propiedades
  • Número fijo de ensayos n
  • Ensayos independientes
  • Probabilidad de éxito constante p
  • Dos resultados posibles
Variable aleatoria

X = número de éxitos en n ensayos

Función de probabilidad
P(X = k) = C(n,k) · p^k · (1−p)^(n−k)
Ejemplo simple

Lanzar una moneda 5 veces. n = 5, p = 0.5

5. Distribución Binomial Negativa

¿Qué modela?

El número de intentos necesarios para obtener un número fijo de éxitos.

Propiedades
  • Ensayos independientes
  • Probabilidad constante p
  • Se busca alcanzar r éxitos
Variable aleatoria

X = número de intentos hasta lograr r éxitos

Función de probabilidad
P(X = k) = C(k−1, r−1) · p^r · (1−p)^(k−r)
Ejemplo simple

Lanzar una moneda hasta obtener 3 caras. r = 3, p = 0.5
